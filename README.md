# ChatGPT Exporter

Firefox extension to export ChatGPT conversations to Markdown files.

## What it does

This extension allows you to export your ChatGPT conversations from the browser to Markdown files on your local machine. Simply navigate to a ChatGPT conversation, click the extension icon, and download the conversation as a `.md` file.

## Installation (Firefox Temporary Add-on)

This extension is designed for use only on `chatgpt.com`.

### Steps

1. Open Firefox
2. Navigate to `about:debugging`
3. Click "This Firefox"
4. Click "Load Temporary Add-on..."
5. Navigate to and select the `manifest.json` file in this extension's root directory
6. The extension will be loaded and available for immediate use

## Usage

1. Open a ChatGPT conversation at `https://chatgpt.com/`
2. Click the ChatGPT Exporter extension icon in your toolbar
3. Click the "Export" button in the popup
4. The conversation will be downloaded as a Markdown file

## Scope

This extension is designed exclusively for use with **chatgpt.com**. No other websites or domains are supported or accessed. This is a chatgpt.com-only extension.

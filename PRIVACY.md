# Privacy Statement

## Data Collection and Storage

This extension is designed to operate with complete user privacy:

- **Local processing only**: All conversation extraction and Markdown rendering happens entirely within your browser. No data is sent to any remote servers.
- **No telemetry**: This extension does not collect or transmit any usage statistics, analytics, or telemetry data.
- **No remote upload**: Exported conversations are saved directly to your local device's downloads folder.

## Website Access

This extension requests access to `https://chatgpt.com/*` only for the purpose of:

- Reading the DOM structure of ChatGPT conversations to extract message content
- Initiating file downloads for exported Markdown files

This is a chatgpt.com-only extension. No other websites are accessed by this extension.

## Third-Party Services

This extension does not use any third-party services, APIs, or cloud storage providers.

## Data Retention

No data is stored by the extension. After each export operation, the extracted conversation data exists only temporarily in browser memory before being written to a local Markdown file.
